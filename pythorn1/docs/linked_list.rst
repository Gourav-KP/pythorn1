Linked List
===========


Quick Start Guide
------------------


.. code-block:: python

    # import the required data structure
    >>> from pythorn.data_structures.linked_list import SinglyList

    # creating linked list
    >>> a = SinglyList()

    # inserting element
    >>> a.insert(5)
    >>> a.insert(8)
    >>> a.insert(16)

    # displaying elements
    >>> a.display_list()
    16 8 5 

    # delete element
    >>> a.delete()
    16

    # size of the linked list
    >>> a.size()
    2
    >>> a.display_list()
    8 5
